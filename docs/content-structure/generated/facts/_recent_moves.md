# Facts - recent moves (rolling)

Running tally of the shapes used at the three positions a reader remembers (reframe opener,
closing line, story pivot), so topic-finding (step 1) can steer a new batch away from the
most-repeated moves, hardest against the highest counts. Counts increment across batches; add a
line only for a genuinely new shape. "Recent batches" keeps the last ~10, older ones dropped.
See CROSS_POST_VARIANCE.md for the format.
